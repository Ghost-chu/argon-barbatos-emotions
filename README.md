# argon-barbatos-emotions
适用于 Wordpress 的 Argon 主题的 原神-巴巴托斯 表情包插件


## 安装
下载插件压缩包，在 WordPress 仪表盘 -> 插件 -> 安装插件 -> 上传插件，上传本插件压缩包并启用，即可注册表情。
